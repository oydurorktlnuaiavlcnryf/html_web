# Personal Portfolio Website

This project is a simple and beautiful personal portfolio website built with HTML and CSS.  
It showcases your profile, skills, and contact information in an attractive layout.

## Features

- Responsive and modern design
- Easy to customize profile information and skills
- Simple structure for beginners to learn HTML and CSS

## How to Use

1. Download or clone this repository.
2. Open `index.html` in your browser to see the website.
3. Edit the HTML and CSS files to add your own information and style.
